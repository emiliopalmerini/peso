## Template ADR

Ogni ADR segue questo formato:

```markdown
# ADR-XXXX: Titolo

## Status

Accepted | Proposed | Deprecated | Superseded

## Context

Descrizione del problema da risolvere

## Decision

Decisione presa e motivazioni

## Consequences

Implicazioni positive e negative
```
